# PushBox
## 项目概述
这是一个基于格子系统的推箱子游戏Demo，主要功能包括角色移动、箱子推拉、冰面滑行、关卡编辑和关卡管理。

### 已完成功能
- **角色移动与推箱子逻辑**  
  - 使用 WASD 控制角色移动。  
  - 当所有箱子移动到 Goal 位置时触发关卡成功判定。  
  - 开门判定暂未想好如何实现，所以完成关卡后会直接进入下一关。

- **冰面逻辑**  
  - 角色和箱子在冰面上可以滑行。

- **关卡搭建**  
  - 已搭建 3 个关卡，参数保存在 `SavedLevel_*.json` 文件中。  

### 操作说明
- **移动**：W/A/S/D  
- **推箱子**：在箱子侧面按方向键即可推动  

### 关卡编辑模式（Runtime）
- 按键选择物体类型：
  - `1` → 墙壁  
  - `2` → 箱子  
  - `3` → Goal  
  - `4` → 冰面
- 放置方式：选择后 **左键点击放置**；**右键单击取消**  
- 编辑中物体颜色：
  - 绿色 → 可放置  
  - 红色 → 不可放置
- 虚拟立方体跟随鼠标显示选择位置  
- **右键双击** → 删除当前位置除地面外的所有物体  
- 快捷键：
  - `Q` → 存储当前关卡  
  - `E` → 载入当前关卡（会清空原有关卡）  
  - `C` → 清空当前关卡

### 文件说明
- `SavedLevel_{CurrentLevel}.json`  
  - 保存每个关卡的参数，包括墙壁、箱子、Goal、冰面及出生点信息  
  - 格式直观，可直接修改关卡数据
